# 一、流体力学全域统一底层方程——Navier-Stokes（N-S）方程组

Navier-Stokes方程组是描述所有流体（气体、液体）运动的全域统一理论，是流体力学的核心公理，不分工况、不做删减，可覆盖星舰上升、回收全流程的所有流动现象。其本质是将牛顿第二定律应用于流体运动，结合质量守恒定律推导得出，完整捕捉了流体的粘性、压缩性、非定常性、湍流等所有物理效应。

## 1.1 完整N-S方程组（三维、可压缩、非定常、粘性）

方程组包含质量守恒方程（连续性方程）和动量守恒方程，结合状态方程可完整描述流体运动，具体形式如下：

$\begin{cases}
\frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \boldsymbol{V}) = 0 \quad \text{（质量守恒方程/连续性方程）} \\[6pt]
\rho \frac{D\boldsymbol{V}}{Dt} = -\nabla p + \mu \nabla^2 \boldsymbol{V} + \rho \boldsymbol{g} \quad \text{（动量守恒方程）}
\end{cases}$

## 1.2 各物理项含义解析

- $\rho$：流体密度（单位：kg/m³），反映空气稀薄程度，星舰低空飞行时密度大，高空时密度显著减小；

- $t$：时间（单位：s），对应非定常运动（如星舰姿态快速翻转、发动机点火制动等瞬态过程）；

- $\boldsymbol{V}$：流体速度矢量（单位：m/s），包含来流速度与星舰运动速度的相对速度（参考系不同，表述不同，详见补充解答）；

- $\frac{\partial \rho}{\partial t}$：密度随时间的变化率，体现流体的非定常特性；

- $\nabla \cdot (\rho \boldsymbol{V})$：质量通量的散度，反映流体的压缩性（密度变化导致的质量迁移）；

- $\frac{D\boldsymbol{V}}{Dt}$：流体质点的加速度（物质导数），包含局部加速度和迁移加速度，体现流体的惯性；

- $p$：流体压强（单位：Pa），是产生气动力（压差阻力、升力）的核心来源；

- $\mu$：流体动力粘度（单位：Pa·s），反映流体的粘性特性，决定摩擦阻力大小；

- $\nabla^2 \boldsymbol{V}$：速度的拉普拉斯算子，体现粘性对流体运动的耗散作用；

- $\rho \boldsymbol{g}$：流体所受重力，星舰低空飞行时需考虑，高空可忽略。

**注**：上述动量方程中 $\mu \nabla^2 \boldsymbol{V}$ 为不可压缩形式的粘性项简化写法。完整可压缩N-S方程的粘性应力张量为 $\nabla \cdot \boldsymbol{\tau}$，其中 $\boldsymbol{\tau}$ 包含剪切粘度μ和体积粘度（第二粘度）λ，可压缩流动中体积粘度项不可忽略。本文为简化展示采用不可压缩形式，不影响后续各阶段截断简化的逻辑一致性。

## 1.3 N-S方程组的核心特性与工程局限

N-S方程组是全域统一的“真理方程”，可描述星舰所有飞行阶段的流动现象（低速、跨音速、大迎角分离、悬停等），但存在两大致命工程局限：

1. 方程组为强非线性耦合偏微分方程，无解析解，无法手动推导计算；

2. 直接求解需捕捉所有尺度的湍流、粘性效应，需亿级超细网格+大型超算集群，计算成本极高、耗时极长（单工况可达数十小时至数天），无法用于日常工程仿真和轻量化计算。

因此，工程实践中需根据具体工况，对N-S方程组进行合理截断、简化，导出各类近似模型，在精度、成本、效率之间实现平衡——这也是星舰各飞行阶段选用不同近似模型的核心原因。

# 二、边界条件、惯性系与参考系转换（补充解答）

结合前文流体力学模型及星舰飞行场景，针对“流体中物体受力与边界条件的关系”“星舰作为惯性系、风动物体不动”两大核心疑问，逐一解答，明确工程仿真中的常规处理逻辑，与前文模型选用内容保持一致。

## 1. 流体中物体受力与边界条件的核心关系

前文所有流体力学模型（纯势流、修正势流等），本质是通过求解流体运动方程（N-S方程组及简化形式），得到流体对物体的作用力，而**物体本身就是流体运动的核心边界条件**，二者密不可分，具体逻辑如下：

1. 边界条件的核心定义：流体运动受固体壁面（星舰壳体、舵面等）限制，无法穿透固体物体，因此“物体表面”就是流体运动的边界，该边界的约束条件（即边界条件）是求解流体方程、计算物体受力的前提。

2. 星舰场景的核心边界条件：前文反复提及的“物面法向速度=0”（$\boldsymbol{V}_{\text{total}} \cdot \boldsymbol{n} = 0$），就是将星舰作为边界条件的具体体现——星舰壳体为固体壁面，流体在壁面处的法向速度必须与星舰表面法向速度一致（若以星舰为惯性系，星舰静止，则流体法向速度为0），这一约束条件直接代入流体方程，才能求解出流体的压强、速度分布，进而通过压强积分得到星舰所受气动力（如压差阻力、升力）。

3. 总结：物体（星舰）是流体运动的“边界”，边界条件（物面法向速度约束）是连接流体方程与物体受力的关键桥梁；求解流体方程时，必须将星舰的外形、姿态作为边界条件代入，才能得到流体对星舰的作用力——这也是前文线元法中“离散星舰壳体线元、施加壁面边界条件”的核心原因。

## 2. 惯性系设定：以星舰为惯性系，“风动物体不动”的工程简化逻辑

流体力学仿真中，参考系的选择不影响最终受力结果（满足伽利略相对性原理），工程上为简化计算，通常采用“以星舰为惯性系，星舰静止、气流运动”的处理方式，具体适配星舰场景的逻辑如下：

1. 星舰惯性系的合理性：星舰在上升、回收各阶段（除极端快速机动外），加速度较小，可近似视为**惯性系**（满足惯性系“不受外力或合外力为零”的近似条件）；即使存在轻微加速度，工程上也可通过添加惯性力修正，不影响核心计算逻辑。

2. “风动物体不动”的本质：这是工程仿真中常用的“相对运动简化”，核心是将“星舰在静止空气中运动”转换为“星舰静止，气流以相同大小、相反方向流过星舰”，具体对应星舰各阶段：
        
    - 阶段1（低空爬升）：星舰竖直向上运动，转换为“星舰静止，气流竖直向下流过星舰”；

    - 阶段4（水平腹拍滑翔/belly flop）：星舰横向水平下落，转换为“星舰静止，气流横向流过星舰（与下落方向相反）”；

    - 阶段6（悬停）：星舰静止，气流仅受星舰姿态影响，无明显相对运动，直接按静止边界条件求解。

3. 简化的核心目的：将“物体运动、流体静止”的非定常问题，简化为“物体静止、流体运动”的准定常问题，大幅降低计算难度——前文所有线元法、势流模型，均基于此简化逻辑，这也是为什么模型中“来流速度”本质是星舰与空气的相对速度，与参考系选择无关。

4. 与前文模型的衔接：前文各阶段模型中提及的“来流速度”“气流分离”“压差分布”，均是基于“星舰为惯性系、星舰静止、气流运动”的简化得到的；例如阶段4水平腹拍滑翔，模型中“气流流过星舰壳体产生分离”，本质就是星舰横向水平下落时，空气相对星舰运动的真实流动效果，二者完全等价。

## 3. 补充说明（贴合工程实际）

1. 若星舰处于极端机动状态（如快速翻转、剧烈俯仰），加速度较大，不再适合作为惯性系，此时需切换为地面惯性系，求解“星舰运动+流体运动”的耦合问题，但星舰回收全6个阶段（不含极端机动），以星舰为惯性系的简化完全适用；

2. 边界条件的设定与参考系选择一致：以星舰为惯性系时，边界条件为“星舰表面流体法向速度=0”；以地面为惯性系时，边界条件为“星舰表面流体法向速度=星舰表面法向运动速度”，二者可通过相对运动转换，最终计算出的星舰气动力完全相同；

3. 前文线元法中“离散星舰壳体、计算各线元压强”，本质就是将星舰作为边界条件，在“星舰静止、气流运动”的简化下，求解流体对各线元的作用力，再叠加得到星舰总气动力，与本次补充的核心逻辑完全一致。

# 四、栅格舵专项（仅归属SuperHeavy一级）
